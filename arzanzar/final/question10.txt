Given an array A of length n of int and a "target" , we need to find number of tuples (x,y,z) such that x,y,z belongs to A and x+y+z=target. (duplication of tuples is allowed).

function cubicAlgo(array A, int n, int target){

   int count=0;

    for(i=0;i<n;i++){

         for(j=0;j<n;j++){

              for(k=0;k<n;k++){

                       if((A[i]+A[j]+A[k])==target){

                              count++;

                       }

            }

        }

  }

return count;




}




The worst case complexity of above program is O(n^3);
